implot(image, figsize=15, 13, cmap='gray_r', scale=0.5, colorbar=False, header=None, wcs=None, **kwargs)[source]¶

Plot an astronomical image, setting default options and easy tweaking of parameters

Parameters

  • image (array_like) – 2D array containing an astronomical image to be plotted. Cutouts should be input as cutout.data.

  • figsize (tuple, optional) – figure size to use. Default: (15,13)

  • cmap (str, optional) – Colormap to use for the image. Default: ‘gray_r’

  • scale (float, optional) – By default, function will scale image to some number of standard deviations about the mean pixel value. Scale sets this number (or fraction). Default: 0.5.

  • colorbar (bool, optional) – Whether to add a colorbar or not. Default: False

  • header (dict, optional) – If input, function will attempt to create a WCS object from header and plot in celestial coordinates. Default: None

  • wcs (WCS object) – If input, the function will plot using a projection set by the WCS. Default: None

  • **kwargs – Additional arguments are passed to matplotlib plotting commands. Currently supported: vmin, vmax.

Returns

figure and axes objects containing currently plotted data.

Return type

fig, ax
